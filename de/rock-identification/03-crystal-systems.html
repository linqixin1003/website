🔬 Grundlagen der Kristallstruktur
Alle Kristalle sind aus sich wiederholenden dreidimensionalen Mustern aufgebaut, die Einheitszellen genannt werden. Die Form und Symmetrie dieser Einheitszellen bestimmt das Kristallsystem:
📐 Schlüsselkonzepte
  • Gitter: Das dreidimensionale Punktgerüst
  • Einheitszelle: Die kleinste sich wiederholende Einheit
  • Symmetrieelemente: Rotationsachsen, Spiegelebenen, Inversionszentren
  • Kristallflächen: Äußere Oberflächen, die die innere Struktur widerspiegeln
🔷 Die sieben Kristallsysteme
Alle Kristalle gehören zu einem von sieben Kristallsystemen, definiert durch die Längen und Winkel ihrer Einheitszellkanten:

🔳 Kubisches System

Parameter: a = b = c, α = β = γ = 90°

Punktgruppen: 23 (T), m3̄ (Th), 432 (O), 4̄3m (Td), m3̄m (Oh)

Kristallographische Formen: Würfel {100}, Oktaeder {111}, Rhombendodekaeder {110}, Tetraeder {111} in Td

Beispiele: Pyrit (Pa3̄), Halite (Fm3̄m), Diamant (Fd3̄m), Granat (Ia3̄d)

🔺 Tetragonales System

Parameter: a = b ≠ c, α = β = γ = 90°

Punktgruppen: 4 (C4), 4̄ (S4), 4/m (C4h), 422 (D4), 4mm (C4v), 4̄2m (D2d), 4/mmm (D4h)

Kristallographische Formen: Tetragonale Prismen {100}, {110}, Tetragonale Bipyramiden {101}, {111}

Beispiele: Zirkon (I41/amd), Rutil (P42/mnm), Anatas (I41/amd)

🔶 Orthorhombisches System

Parameter: a ≠ b ≠ c, α = β = γ = 90°

Beispiele: Olivin, Pyroxen, Baryt

🔷 Hexagonales System

Parameter: a = b ≠ c, α = β = 90°, γ = 120°

Beispiele: Quarz, Beryll, Apatit

🔸 Trigonales System

Parameter: a = b = c, α = β = γ ≠ 90°

Beispiele: Calcit, Hämatit, Turmalin

🔹 Monoklines System

Parameter: a ≠ b ≠ c, α = γ = 90° ≠ β

Beispiele: Gips, Orthoklas, Hornblende

🔻 Triklines System

Parameter: a ≠ b ≠ c, α ≠ β ≠ γ ≠ 90°

Beispiele: Plagioklas-Feldspat, Türkis

💎 Kristallhabitus
Kristallhabitus beschreibt die allgemeine Form und das Aussehen von Kristallen:

📏 Prismatischer Habitus

Lange, säulenartige Kristalle. Häufig bei Quarz, Turmalin und Beryll.

📋 Tafelig

Flache, plattenartige Kristalle. Typisch für Baryt und Wulfenit.

🧊 Würfelig

Gleichmäßig entwickelte Kristalle. Charakteristisch für Pyrit und Halite.

🌿 Dendritisch

Baumähnliche Wachstumsmuster. Häufig bei Mangan-Oxiden und nativem Kupfer.

🔍 Bestimmungstipps
  • Messen Sie Winkel zwischen Kristallflächen mit einem Goniometer
  • Beachten Sie Zwillingsbildung und Verwachsungen
  • Dokumentieren Sie Flächenentwicklung und relative Größen
  • Berücksichtigen Sie Wachstumsbedingungen und Umgebung
