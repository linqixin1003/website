変成転換

岩石の完全な変化プロセス
変成転換は既存岩石が新しい鉱物学的・組織的特徴を獲得する過程です。この変化は固体状態で起こり、原岩の完全な変質をもたらします。
🔄 変成過程の種類

プログレード変成作用

温度・圧力の増加による前進変成作用。新しい高温・高圧鉱物の形成。典型的な造山帯の主変成過程。

レトログレード変成作用

温度・圧力の減少による後退変成作用。高変成度鉱物の低変成度鉱物への変化。上昇と冷却過程で発生。

⚗️ 変成反応

連続反応

鉱物組成の段階的変化。例：黒雲母 + 石英 → カリ長石 + ざくろ石 + H₂O。温度上昇とともに段階的に進行。

不連続反応

特定のP-T条件での急激な鉱物変化。例：カオリナイト → パイロフィライト + 石英 + H₂O。明確な反応境界を持つ。

💎 組織発達
  • 再結晶： 粒径増大と結晶完形度向上
  • 片理形成： 鉱物の優選配列による面構造
  • 線構造形成： 鉱物の線状配列
  • ポーフィロブラスト成長： 大型変成鉱物の形成
🌍 原岩からの変化

泥質岩の変成

頁岩 → スレート → 千枚岩 → 片岩 → 片麻岩。粘土鉱物から雲母、そして長石・石英への段階的変化。

砂岩の変成

砂岩 → 珪岩。石英粒子の再結晶と粒界の融合。元の堆積構造の消失。

石灰岩の変成

石灰岩 → 大理石。方解石の再結晶と粒径粗粒化。純度により様々な変成鉱物の形成。

火成岩の変成

花崗岩 → 片麻岩、玄武岩 → 角閃岩。原鉱物の新しい変成鉱物への置換。

🔬 変成帯と等変成度線

ザクロ石帯

中変成度の指標。ざくろ石 + 黒雲母 + 白雲母 + 石英 + 斜長石の集合。500-600°C、0.4-0.8 GPa。

十字石帯

高変成度の指標。十字石 + ざくろ石 + 黒雲母 + 白雲母の集合。550-650°C での安定域。

🎯 変成相系列
地温勾配の違いにより、低圧型（アンダルサイト系列）、中圧型（カイアナイト系列）、高圧型（藍晶石系列）に分類。
⏰ 変成作用の時間尺度
  • 接触変成： 数百年～数千年の急速な変化
  • 広域変成： 数百万年～数千万年の長期過程
  • 反応速度： 拡散と流体の役割が重要
  • 平衡到達： 部分平衡から完全平衡へ
💧 流体の役割

変成流体

主にH₂O、CO₂。鉱物反応の触媒として機能。イオン輸送と反応速度を促進。

流体源

脱水反応、マグマ脱ガス、地下水、海水。変成過程での流体生成と移動。

🌍 地質学的意義
  • 造山過程： 大陸衝突と地殻肥厚の記録
  • 地殻進化： 古地温勾配と熱構造の推定
  • 資源形成： 変成鉱床と工業鉱物の生成
  • 惑星科学： 他天体での変成過程の理解
結論： 変成転換は地殻における最も重要な岩石変化過程の一つです。この過程の理解は地球深部の条件と過程を解明する鍵となります。
