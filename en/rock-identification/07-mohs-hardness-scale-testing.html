ğŸ¯ Testing Technique
Proper technique is essential for accurate hardness determination:
ğŸ”¬ Professional Method
  1. Select fresh, unweathered surface
  2. Apply firm, controlled pressure
  3. Draw testing tool across mineral surface
  4. Examine for actual scratch (not powder residue)
  5. Test both directions - mineral scratching tool vs. tool scratching mineral
  6. Clean surfaces between tests
âš ï¸ Common Errors
  • Mistaking powder residue for scratches
  • Testing on weathered or altered surfaces
  • Using insufficient pressure
  • Not testing reciprocal hardness
  • Ignoring crystal face variations
