🍄 소개 : 버섯 사진이 심각한 이유가있는 이유
태평양 북서부의 고대 숲에서 완벽한 유독 한 파리 아 마니 타를 처음 발견했을 때, 그 순간은 버섯 사진에 대한 나의 이해를 영원히 바꿨습니다.밝은 빨간 구근 덮개에는 동화 장면과 같이 흰색 비늘이 점재되어 있습니다. 그러나 경험이 풍부한 진균 학자로서 나는 "아름다운"사진이 충분하지 않다는 것을 알고 있습니다.진정으로 귀중한 버섯 사진은 미학과 과학을 모두 고려해야하며 자연의 아름다움을 포착 할뿐만 아니라 충분한 식별 특성을 제공해야합니다.🍄

지난 15 년 동안, 나는 렌즈를 통해 500 개가 넘는 버섯 종을 녹음하여 학업 연구를위한 수천 개의 과학 사진을 제공하고, 사진 녹음의 일반적인 함정을 피하기 위해 수많은 초보자를 안내했습니다.이 안내서는 이러한 경험을 공유하여 예술적이고 과학적인 버섯 사진 레코드를 만드는 데 도움이됩니다.

🍄 사진 레코드의 여러 가치 : 아름다움을 넘어서 실질적인 고려 사항
📌 인증 지원 : 비주얼 노트북

실제 사례 : 지난 가을, 나는 캘리포니아 북부의 레드 우드 숲에서 익숙하지 않은 아 마니 타를 발견했습니다.시스템에서 찍은 7 개의 각도 사진을 통해, 나는 3 명의 전문가와 원격으로 작업 할 수 있었고 마침내 이것이 공식적으로 문서화되지 않은 변형임을 확인했습니다.자세한 사진이 없으면이 발견은 거의 불가능합니다.

주요 장점 :

  • 집으로 돌아온 후에도 계속 식별 : 현장 시간이 제한되어 있으며 사진을 통해 집에서주의 깊게 공부할 수 있습니다.
  • 전문가 상담의 시각적 증거 : 명확한 사진은 수천 단어보다 낫습니다.
  • 참조 자료 비교 : 삽화 또는 온라인 리소스와 정확한 비교
📌 메모리 보조 : 인간 기억의 단점에 대한 구성

인간의 기억은 선택적이고 시간이 지남에 따라 왜곡됩니다.나는 버섯의 냄새와 질감을 분명히 기억한다고 생각했고, 사진과 음표를 비교할 때까지 내 기억이 편견이 있다는 것을 몰랐습니다.

과학적 근거 : 연구에 따르면 사람들은 48 시간 후에 관찰 세부 사항의 70% 정도를 잊어 버립니다.사진은 당시 중요하지 않다고 생각했던 미묘한 기능을 캡처합니다.

🔬 과학적 가치 : 시민 과학자의 기여

전문적인 통찰력 : ITATURALIST와 같은 플랫폼을 사용하여 아마추어 사진은 Mycology를 바꾸고 있습니다.2019 년에 워싱턴 주 등산객의 사진이 새로운 볼 레트를 발견하게되었습니다.귀하의 기록은 예상치 못한 과학적 가치를 가질 수 있습니다.

구체적인 기여에는 다음이 포함됩니다.

  • 종 분포 기록 : 곰팡이에 대한 기후 변화의 영향을 추적하는 데 도움이됩니다.
  • 현상 데이터 : 다른 종의 발생 시간과 지속 시간을 기록하십시오.
  • 잠재적 인 새로운 종 발견 : 전문가들은 사진을 통해 알려지지 않은 종을 식별합니다.
📌 법적 및 보안 증거 : 중요한 순간에 생명을 구하는 정보

실제 이야기 : 2018 년 콜로라도의 한 가족은 실수로 독이있는 버섯을 먹었고, 그들의 사진은 독극물 통제 센터가 몇 분 안에 종을 결정하고 올바른 치료를 안내하는 데 도움이되었습니다.

법적 수준에서 명확한 사진은 다음과 같습니다.

  • 중독 유형을 정확하게 식별하고 의학적 치료를 안내합니다.
  • 권리 분쟁을위한 증거 제공
  • 컬렉션의 위치 및 권한을 기록하십시오
🍄 여덟 필수 촬영 각도 : 녹음을 완료하는 키
📌 1. 현장 성장 사진 : 환경은 이야기를 들려줍니다.

중요한 이유 :

버섯은 환경에서 분리 할 수 ​​없습니다.많은 종이 특정 나무 종과의 균근 관계를 형성하며, 성장 매트릭스 (토양, 목재 종)는 식별의 핵심 단서입니다.

전문 기술 :

  • 1 미터 이상의 환경을 포함하여 2 단계 뒤로 물러서
  • 버섯과 기질 사이의 연결을 포착합니다
  • 주변 식물과 나무가 들어 있습니다
  • 자연광 조건을 기록하십시오

일반적인 오류 : 버섯 클로즈업에 너무 많이 집중하고 환경 배경을 무시하고 생태 관련 정보를 잃습니다.

📌 2. 모자의 상단 : 정체성의 첫인상

주요 기능 캡처 :

  • 덮개 모양 : 원추형, 볼록, 평평한 모양 등
  • 색상 및 분포 : 중심과 가장자리의 색상 차이에주의를 기울이십시오.
  • 표면 질감 : 끈적 끈적하고 매끄럽고 건조, 비늘 등
  • 가장자리 특성 : 경사, 줄무늬 등

촬영 기술 :

  • 45도에서 촬영, 측면의 상단과 일부를 모두 보여줍니다.
  • 중앙에서 가장자리까지 초점이 명확 해지십시오.
  • 자연 산란 빛을 사용하여 강한 그림자를 피하십시오
📌 3. 표지의 바닥 : 포자의 비밀 세계

이것이 가장 중요한 식별 특성 중 하나 인 이유 :

곰팡이 주름, 구멍 또는 치아와 같은 구조는 버섯 분류의 주요 기초입니다.그들의 색상, 밀도 및 애착은 결정적인 증거를 제공합니다.

전문 방법 :

  • 손상을 피하기 위해 직접 뒤집는 대신 버섯을 부드럽게 기울여
  • 만지지 않고 작은 거울을 사용하여 아래에서 반사
  • 턱과 줄기가 어떻게 연결되어 있는지 명확하게 표시하십시오.
  • 특성이 가장 분명한 박테리아 프릴의 가장자리에 집중
📌 4. 측면 프로파일 : 전반적인 형태

주요 정보 :

  • 표지 섹션 모양
  • 줄기의 형태와 비율
  • 박테리아 고리의 위치와 특성
  • 각 부분의 비례 관계

사진 제안 :

  • 버섯과 같은 수평선에서 촬영
  • 간단한 배경을 사용하여 개요를 강조 표시하십시오
  • 전체가 급격히 초점을 맞추고 있는지 확인하십시오
📚 5. 줄기 및 기본 : 숨겨진 식별 키

전문가들은 다음과 같이 제안합니다. 이것은 가장 자주 간과되지만 종종 가장 중요한 각도입니다.많은 치명적인 유독 버섯의 식별 특성이 기본에 있습니다.

캡처 해야하는 기능 :

  • 위 표면 질감 : 매끄럽고 망상, 비늘?
  • 박테리아 반지 : 움직일 수 있는지 여부, 위치, 색상
  • 박테리아 : 존재 여부, 모양, 색상
  • 기본 모양 : 확대 여부에 관계없이, 거짓 뿌리가 있는지 여부

운영 안내서 :

  • 버섯을 손상되지 않지만 조심스럽게 파헤쳐베이스를 그대로 유지하십시오.
  • 먼지를 부드럽게 제거하지만 키 구조를 유지하십시오
  • 기본 기능을 닫고 명확하게합니다
📌 6. 수직 섹션 : 내부 구조 공개

필수 불가능한 이유 :

내부 색상, 질감 및 구조는 Lactobacillus와 같은 많은 속의 결정적인 특성입니다.

전문 기술 :

  • 날카로운 칼을 사용하여 청소를 잘라냅니다
  • 지금 사진을 찍으려면 초기 색상을 기록하십시오.
  • 색상 변경을 기록하기 위해 5-15 분 동안 기다리십시오
  • 흰색 배경으로 촬영하고 대비를 향상시킵니다
  • 버섯 고기와 줄기의 두께가 속이 비어 있는지 여부에주의하십시오.
📌 7. 스케일 사진 : 크기의 중요성

과학적 근거 : 참조 대상이 없으면 인간의 두뇌가 크기를 정확하게 판단하기가 어렵습니다.척도는 객관적인 데이터를 제공합니다.

모범 사례 :

  • 센티미터/mm 규모의 눈금자를 사용하십시오
  • 동전은 대안으로 사용할 수 있지만 통치자는 더 전문적입니다.
  • 스케일이 버섯과 같은 비행기에 있는지 확인하십시오.
  • 적어도 하나의 사진에는 명확한 규모가 포함되어 있습니다
📌 8. 포자 인쇄 : 최종 식별 증거

전문적인 통찰력 : 포자 색상은 Amanita와 같은 많은 속에서 식별의 결정적인 기능입니다.

제작 및 촬영 가이드 :

-뚜껑을 반 흰색과 반 검은 종이에 놓습니다

  • 공기 흐름을 방해하지 않도록 그릇이나 상자를 덮으십시오.
  • 완전한 포자 인쇄를 얻으려면 2-24 시간을 기다립니다.
  • 깜박이는 것을 피하기 위해 자연광으로 촬영하십시오
  • 태그 정보 포함 (날짜, 장소, 번호)
🛠️ 전문가 등급 사진 장비 : 스마트 폰에서 전문 카메라에 이르기까지
🔧 스마트 폰 : 주머니에 강력한 도구

최신 휴대 전화 기능 평가 :

최신 플래그십 폰은 과학적 등급의 사진을 좋은 빛으로 만들 수 있습니다. iPhone 14 Pro 및 Samsung Galaxy S23 Ultra와 같은 장치의 매크로 모드는 인상적입니다.

모바일 사진 장점 :

  • 사용할 준비가되면 우발적 인 발견이 놓치지 않습니다
  • 통합 GPS 자동 위치 기록
  • 즉시 공유 및 업로드 기능
  • 다양한 전문 사진 응용 프로그램

모바일 사진 제한 :

  • 거시적 기능에는 여전히 물리적 인 한계가 있습니다
  • 저조도 환경에서 상당한 소음
  • 센서 크기 제한 동적 범위

휴대 전화 전문가 설정 :

  • 기본 카메라 응용 프로그램을 사용하기위한 전문 모드
  • 자동 플래시를 끄십시오
  • 그리드 라인 보조 구성을 활성화합니다
  • 원시 형식 사용 (지원되는 경우)
📌 특수 카메라 시스템 : 궁극적 인 화질을 추구합니다

업그레이드 할 때 : 열심히 공부하기 시작하거나 출판 등급 화질이 필요할 때 전용 카메라에 투자하는 것이 합리적인 선택입니다.

시스템 선택 안내서 :

  • 마이크로 싱글 카메라 : 가볍고 강력하며 실외 용에 적합합니다.

-SLR 카메라 : 광학 뷰 파인더 장점, 긴 배터리 수명

  • 주요 기능 : 교환 가능한 렌즈, 수동 제어, 원시 형식

권장 구성 :

  • 바디 : Sony A7 시리즈, Canon R 시리즈, Nikon Z 시리즈
  • 매크로 렌즈 : 90-105mm f/2.8 매크로 렌즈는 금 표준입니다.
  • 백업 배터리 및 메모리 카드 : 야생에서 충전 할 수 없습니다.
📌 필수 보조 장비 : 사진 품질 향상의 열쇠

삼각대 시스템 :

  • 데스크탑 삼각대 : 낮은 각도 촬영에 적합합니다
  • 전체 크기 삼각대 : 저조도 및 매크로 용
  • 휴대폰 클립 어댑터 : 전화기를 삼각대에 수정

조명 솔루션 :

  • 링 LED 조명 : 균일 한 매크로 조명
  • 작은 반사기 : 음영 영역에 자연광을 반사하십시오
  • 오프 머신 플래시 : 소프트 라이트 커버와 함께 사용됩니다

전문 액세서리 :

  • 포커스 트랙 프레임 : 초점 초점 오버레이 용
  • 필드 배경 보드 : 접는 흰색/검은 색/회색 배경
  • 스케일 바 및 컬러 카드 : 과학 기록을위한 필수품
🛠️ 실제 촬영 기술 : 초보자부터 전문가까지
📌 초점 및 필드 마스터리 심도

현장 이해 심도 :

거시 사진에서, 피사계 심도는 밀리미터로 측정됩니다. F/16은 몇 밀리미터의 명확한 범위 만 제공 할 수 있습니다.

실제 전략 :

  • F/8-F/16을 사용하여 식별 사진을 위해 최대 피사계 깊이를 얻으십시오.
  • 예술적 사진 F/2.8-F/5.6을 사용하여 배경 흐림을 만듭니다
  • 가장 중요한 식별 기능에 중점을 둡니다

고급 팁 : 초점 오버레이

1. 삼각대에서 카메라를 고정하십시오

2. 가장 가까운 곳에서 가장 먼 지점까지 여러 장의 사진을 찍습니다.

3. Helicon Focus 또는 Photoshop을 사용하여 종합합니다

4. 앞뒤로 완전히 명확한 이미지를 얻으십시오.

📌 조명 제어 : 자연과 인공의 균형

최고의 자연광 조건 :

  • 흐린 날 : 부드럽고 심지어 단단한 그림자가 없습니다
  • 프라임 타임 : 일출 후 및 일몰 1 시간 전
  • 그늘 아래 : 얼룩덜룩 한 반점을 피하십시오

인공 조명 기술 :

  • 디퓨저를 사용하여 플래시를 부드럽게합니다
  • 직접 조사를 피하기 위해 LED 조명 각도 조정
