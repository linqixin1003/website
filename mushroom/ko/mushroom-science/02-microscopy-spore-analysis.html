02 버섯 식별의 현미경 세계

Professional Mycology Guide
🔬 Mushroom Science 📖 11 minute read 🟡 Intermediate

현미경으로 버섯 식별은 과학뿐만 아니라 예술이기도합니다.버섯 컬렉션을 진지하게 받아들이는 야외 애호가에게 마이크로 식별 기술을 마스터하는 것은 유사한 종에서 식용 종을 정확하게 식별하고 잠재적 위험을 피하는 것을 의미합니다.이 기사는 미세한 기능을 통해 버섯 식별 기술을 향상시키는 방법을 심도있게 탐구합니다.🍄
🎯 왜 마이크로 식별이 중요한가

거시 식별의 한계

야생에서는 많은 버섯 종들이 매우 비슷해 보이지만 식용과 독성은 크게 다를 수 있습니다.Amanita 속을 예를 들어, 일부 치명적인 독사 버섯은 거시적 방식으로 식용 종들과 거의 구별 할 수 없습니다.마이크로 특징은 특히 비슷한 외관을 가진 종에 대해 의심의 여지가없는 기초를 제공합니다.

실제 사례 : Amanita 식별 도전

나는 한때 미국의 태평양 북서부에서 전형적인 사건을 겪었습니다. 두 명의 숙련 된 수집가는 흰 버섯이 식용인지에 대해 의견이 일치하지 않습니다.거시적, 그것은 식용 Amanita Lanei처럼 보이지만 현미경에서 포자 분석은 실제로 치명적인 천사 버섯 (Amanita bisporigera)임을 보여줍니다.이 예제는 미세한 식별이 생명을 어떻게 절약하는지 생생하게 보여줍니다.

과학적 기초

곰팡이 분류는 현미경 특징에 점점 더 의존하고 있습니다.포자 형태, 낭성 구조 및 균사적 특성은 비교적 안정적이고 환경 적 요인의 영향을받지 않는 신뢰할 수있는 분류 학적 데이터를 제공합니다.

🛠️ 전문 장비 안내서 : 곰팡이 실험실 구축

현미경 선택 전략

복합 광학 현미경은 핵심 도구이며 배율은 400-1000 회 범위에 있어야합니다.초보자에게는 기본적인 요구를 충족시키기 위해 $ 300-500 사이의 요금 수준의 바이오 미스 스코프를 권장합니다.보다 전문적인 연구에는 오일 거울과 고급 광학 시스템이 장착 된 2,000 달러 이상의 모델이 필요할 수 있습니다.

전문가 권장 사항 : ABE 콘덴서와 기계적 단계가있는 모델을 찾으십시오. 작은 구조를 관찰 할 때 매우 중요합니다.

필수 액세서리 목록

  • 슬라이드 및 커버 슬립 : 다른 현미경에 맞게 다중 두께 (No.1 및 No.1.5)를 준비
  • Stainant Set : Melzer 시약, 콩고 레드, 5% KOH 솔루션 포함
  • 접안 렌즈 마이크로 미터 : 포자 크기를 정확하게 측정하는 데 사용됩니다
  • 해부학 도구 : 해부 바늘, 날카로운 날 (수술 블레이드가 가장 좋습니다)
  • 실험실 노트북 : 기록 관찰 결과에 대해 자세히 설명합니다

빠른 팁 : 처음에 모든 전문 장비를 구입할 필요는 없으며 기본 구성은 가장 일반적인 버섯을 식별하기에 충분합니다.

🔬 포자 인쇄 생산 : 과학과 예술의 조합

최적화 된 포자 인쇄 기술

포자 인쇄는 포자 색상 정보를 제공 할뿐만 아니라 현미경 관찰을위한 많은 수의 포자를 얻습니다. 전통적인 개선에는 다음이 포함됩니다.

1. 성숙하지만 완전히 확장되지 않은 덮개를 선택하십시오

2. 반백색 및 반 흑인 용지를 사용하여 다양한 색상의 포자 관찰을 용이하게합니다.

3. 유리 그릇을 덮으려면 공기 건조를 방지하지만 적당한 공기 순환을 허용하기 위해 작은 간격을 남기십시오.

4. 실온에서 6-24 시간 동안두면 특정 시간은 버섯 유형에 따라 다릅니다.

실용 기술 : 포자를 생산하기 어려운 종의 경우, 용기의 습도를 높이거나 박테리아 캡을 종이 대신 알루미늄 호일에 배치하여 후속 포자 수집을 용이하게합니다.

포자 색상의 분류

포자 색상은 버섯 분류의 중요한 지표입니다.

  • 흰색 : Amanita, Russula 등의 일반적인 색상

-Pink : Pluteus의 특징적인 색상

  • 브라운 : 다양한 색조를 포함하여 많은 다공성 박테리아의 특징입니다.
  • 자주색 갈색 : psilocybe의 전형적인 색상
  • 검은 색 : Coprinus 속 및 기타 속의 특성

전문가 권장 : 자연광에서 포자 색상을 관찰하면 인공 광원이 색상 인식을 바꿀 수 있습니다.

🍄 포자 관찰 : 샘플에서 데이터로

전문 샘플링 기술

포자 인쇄에서 샘플링 할 때 습식 해부 바늘 팁을 사용하여 포자를 부드럽게 담그십시오.박테리아 주름에서 직접 샘플링하는 경우 박테리아 주름 구조를 파괴하지 않고 소량의 포자를 조심스럽게 긁어 모으십시오.다공성 박테리아의 경우, 튜브 웰에서 샘플을 채취 하였다.

사진 최적화 방법

1. 슬라이드 중앙에 작은 증류수 한 방울 또는 5% KOH 용액을 추가하십시오.

2. 액 적에 포자 샘플을 부드럽게 분산시킨다

3. 거품을 피하기 위해 약 45 도의 각도로 커버 슬립을 천천히 낮추십시오.

4. 조직을 사용하여 과도한 액체를 빨아 먹습니다

일반적인 오류 : 너무 많은 샘플을 사용하면 셀 층이 너무 두꺼워지고 집중하기가 어려울 수 있습니다.드문 샘플은 일반적으로 더 명확한 관찰을 제공합니다.

시스템 관찰 프로세스

1. 저전력 미러 (100x)를 사용하여 샘플과 일반 구조를 먼저 찾으십시오.

2. 세부 사항을 관찰하려면 고전력 미러 (400x)로 전환

3. 오일 미러 (1000x)를 사용하여 포자 표면 패턴 및 기타 측면과 같은 미세 구조를 관찰하십시오.

포자 특성 분석

모양 분류

  • 구형 : 일부 Amanita 종과 같은 완전히 둥글다
  • 타원 : 종횡비는 약 1.5 : 1이며, 많은 토마토 벌레에서 일반적으로 발견됩니다.
  • 원통형 : 길이는 너비보다 상당히 큽니다.
  • 아몬드 모양 : 중간에 넓은 양쪽 끝을 가리키기
  • 스핀들 : 중간에 넓고 양쪽 끝에서 얇은

정확한 측정 기술

20-30 성숙 포자를 보정 된 접안 렌즈 현미경을 사용하여 측정하고 길이와 폭을 기록 하였다.상당히 미숙하거나 변형 된 포자를 배제하기 위해 평균과 범위를 계산하십시오.

전문적인 팁 : 포자의 크기는 같은 종 내에서도 다릅니다. 여러 포자를 측정하면보다 정확한 데이터를 얻을 수 있습니다.

표면 패턴 식별

  • 매끄러운 : 표면에 돌출부가 없습니다

-사마귀 같은 : 불규칙한 사마귀 같은 돌기

  • 피어싱 : 스파이크 돌기
  • 망상 패턴 : 망상 패턴

특별 구조 관찰

  • 버드 포어 : 포자 벽의 얇은 영역, 발아 할 때 여기에서 부러집니다.
  • 오일 액 적 : 특정 유형의 식별 가치를 가진 포자의 저장 물질
🍄 cystosome : 무시 된 식별 키

낭성 신체 유형에 대한 자세한 설명

낭포체는 매우 다양한 형태를 가진 박테리아 프릴의 가장자리 또는 측면에 특수한 세포입니다.

-Pleurocystidia : 다양한 모양으로 박테리아 주름의 측면에 위치

-Cheilocystidia : 많은 속의 식별에 중요한 박테리아 주름의 가장자리에 위치하고 있습니다.

실제 적용 : psilocybe 속의 식별은 캡슐의 특성에 매우 의존하며 올바른 식별은 법적 및 안전 문제와 관련이있을 수 있습니다.

관찰 방법의 최적화

박테리아 주름의 얇은 부분을 준비하려면 연습과 인내심이 필요합니다.

1. 날카로운 칼날로 작은 박테리아 조각을 자릅니다.

2. 슬라이드에 놓고 해부 바늘로 부드럽게 분리하십시오.

3. 적절한 매체 추가 (물 또는 koh)

4. 커버 슬립을 덮고 부드럽게 눌러 샘플을 얇게 만듭니다.

전문가 권장 : 양면 블레이드를 사용하여 얇은 슬라이스를 준비하여 관찰 품질을 향상시킵니다.

🍄 균사체 구조 : 곰팡이의 뿌리

myhypha 유형 식별

  • 생식 균사체 : 포자 생산을 담당하는 관절이 잠긴 다이어프램이 있습니다.
  • 골격 균사체 : 두꺼운 벽, 구조적지지를 제공합니다
  • 균사체와 결합 : 얇은 벽, 연결 기능을 수행합니다.

자물쇠 모양의 연합의 중요성

클로닝 된 조합은 특정 Bacidioids의 특징적인 구조이며, 세포의 교차점에서 작은 루프로 나타납니다.그 존재 또는 부재는 중요한 분류 기능입니다.

관찰 기법 : 캡 내부에서 매우 얇은 조직 시트를 찢거나 스토킹하여 KOH 용액에 넣어 관찰하십시오.

🍄 멜체 반응 : 화학 식별 방법

멜저 시약 애플리케이션 안내서

멜체 시약 (요오드 라이드-요오드 용액)을 사용하여 전분 반응을 검출합니다.

  • 전분 반응 : 진한 파란색 또는 자주색 검은 색
  • 비 전속 반응 : 노란색 또는 변하지 않음

-전분 같은 반응 : 붉은 갈색을 돌립니다

실제 응용 프로그램

Russula와 Lactarius 속의 식별은 Melzer 반응에 크게 의존합니다.예를 들어, Russula 포자의 대부분의 종은 전분에 반응하는 반면, Lactarius의 포자는 일반적으로 비 전속입니다.

안전 팁 : 멜저 시약은 독성이 있고, 수술 중에 장갑을 착용하고, 환기가 잘되는 곳에서 수행합니다.

🍄 캡의 표피 구조 : 표면의 비밀

표피 유형 분류

  • 왁스 퀼트 (ixocutis) : 글루텐 층 덮개
  • trichoderm : 직립 균사체로 구성됩니다
  • Palisadoderm : 세포의 수직 배열
  • 상피 : 구형 또는 타원형 세포로 구성됩니다

관찰 방법

캡의 가장자리에서 슬라이스를 자르고 표면 층과 소량의 기저 조직을 포함 시키십시오.콜로이드 층의 경우 KOH가 청소를 도와야 할 수도 있습니다.

🛠️ 전문 현미경 기술 본질

생산을위한 완벽한 7 단계 방법

1. 슬라이드와 커버 슬립을 청소하십시오

2. 올바른 매체 (물, 코 또는 기타)를 선택하십시오.

3. 적절하게 샘플하고 배포하십시오

4. 커버 슬립을 조심스럽게 덮으십시오

5. 기포 제거 (커버 슬립을 탭)

6. 과도한 액체를 빨아들입니다

7. 가장자리에서 밀봉 (장기 저장 용)

염색 기술 최적화

  • 콩고 레드 : 균사체 및 포자 벽의 가시성 향상
  • 면화 블루 : 특히 미세한 패턴을 관찰하는 데 적합합니다

-Melzer 시약 : 포자 또는 조직에 직접 사용됩니다

전문가 팁 : 염색하기 어려운 구조물의 경우 먼저 KOH로 청소 한 다음 염료를 바릅니다.

정확한 측정 프로토콜

1. 각 사용하기 전에 마이크로 미터를 교정하십시오

2. 여러 개의 시야에서 포자를 측정하십시오

3. 최대, 최소 및 평균 값을 기록하십시오

4. 측정 조건 (미디어, 배율)에주의를 기울이십시오.

🔬 기록 및 문서 : 과학적 방법의 적용

시스템 레코드 템플릿

다음을 포함하여 표준화 된 레코드 양식 작성

  • 데이터 수집 (날짜, 위치, 서식지)
  • 거시 경제 특징 설명
  • 상세한 마이크로 피처 레코드
  • 화학 반응 결과
  • 고품질 사진 또는 그림

사진 기술 가이드

최신 현미경 카메라의 가격이 크게 떨어졌으며 스마트 폰 어댑터도 좋은 결과를 얻을 수 있습니다.주요 촬영 기술 :

  • 균일 한 조명을 사용하십시오
  • 다른 초점 평면에서 여러 장의 사진을 찍습니다
  • 스케일 포함
  • 기록 매개 변수를 기록합니다

디지털 도구 응용 프로그램

이미지 분석 소프트웨어는 포자 크기를 측정하고 데이터베이스 관리를 사용한 레코드를 관찰하는 데 사용되었습니다.Inaturalist 및 Mushroom Observer와 같은 플랫폼은 발견을 공유하고 전문가와 의사 소통 할 수 있습니다.

🍄 일반적인 도전과 솔루션

어려움에 집중

  • 이유 : 샘플이 너무 두껍습니다
  • 해결책 : 더 얇은 슬라이스를 준비하고 응축기를 조정하십시오

버블 문제

  • 예방 : 각도에서 커버 슬립을 천천히 낮추십시오
  • 솔루션 : 커버 슬립을 탭하거나 바늘 팁을 사용하여 거품을 안내합니다.

희소 포자

형광 현미경

특정 형광 염료는 기존 현미경에서 관찰하기 어려운 구조를 강조 할 수 있습니다.

위상 대비 현미경

투명한 구조의 대비를 개선하고 염색없이 살아있는 샘플을 관찰하십시오.

🍄 실제 응용 프로그램 시나리오

유사한 종의 정확한 식별

예를 들어, 독성 자이로 미트 라 종과의 식용 모렐 (모렐)을 구별하면 현미경 특성은 결정적인 증거를 제공합니다.

새로운 발견 및 설명

미세한 특징은 새로운 곰팡이 종을 설명하는 데 필수적인 구성 요소입니다.애호가들은 또한 신중한 현미경 관찰을 통해 새로운 종을 발견 할 가능성이 높습니다.

독성 평가

일부 독성 버섯은 갈레 리나의 포자 특성과 같은 독특한 현미경 특성을 가지고 있습니다.

🛠️ 비전문가 애호가를위한 실용적인 조언

현미경은 필수가 아니라 부가 가치 도구

가장 일반적인 식용 버섯은 거시적 특성을 통해 안전하게 식별 할 수 있습니다.현미경은 엔트리 레벨 요구 사항이 아니라 기술과 이해를 향상시키는 도구입니다.

초보자 선택

$ 200-500의 현미경은 대부분의 아마추어 요구에 충분합니다.중고 시장에서도 양질의 장비를 찾을 수 있습니다.

공유 자원

지역 곰팡이 협회, 커뮤니티 칼리지 또는 자연 센터는 현미경 사용 기회를 제공 할 수 있습니다.

기존 자원 사용

인터넷에는 이미 학습 및 비교에 사용할 수있는 많은 고품질 현미경 사진이 이미 있습니다.

1. 기본 장비 : 품질 현미경 및 기본 액세서리에 투자

5. 커뮤니티에 가입 : 다른 애호가들과 상호 작용하고 워크샵에 참여하고 진출합니다.
