04 현장 식별에 대한 실용 가이드

Professional Mycology Guide
🔍 Mushroom Identification 📖 16 minute read 🔴 Advanced

덮개가 버섯의 "외관"이라면, 곰팡이 주름, 구멍 및 기타 포자의 표면은 버섯의 "엔진 챔버"입니다. 곰팡이의 수명을 계속하기 위해 수백만 개의 포자가 생산되고 방출됩니다.이러한 구조는 생물학적으로 중요 할뿐만 아니라 식별에서 가장 귀중한 특징 중 하나입니다.필드 컬렉션에서 25 년의 경험을 가진 마이코 학자로서, 나는 당신에게 명확하게 말할 수 있습니다. 포자 표면 특성에 대한 관찰 기술을 마스터하는 것은 치명적인 담배와 함께 식용 맛있는 음식을 치명적인 담배와 구별하는 데 핵심 기술입니다.🍄
🎯 포자 표면 특성이 왜 그렇게 중요한가요?

초기 컬렉션 경력에서 나는 박테리아 주름의 부착을 무시했기 때문에 유독 한 아 마니 타를 거의 놓쳤다.그 경험으로 인해 버섯 식별은 인식에 근거한 것이 아니라 과학적 특성에 기초한 정확한 판단에 기초한다는 것을 깊이 깨달았습니다.

포자 표면의 생물학적 중요성

포자의 표면은 버섯의 생식 센터입니다.중간 크기의 버섯 캡은 하루에 약 1,600 만 포자를 방출 할 수 있으며, 수명주기 동안 총 포자 수는 수십억에이를 수 있습니다.이 미세한 포자는 풍력에 의해 퍼지고, 적절한 환경에서 균사체로 발아하며, 새로운 수명주기를 시작합니다.

승인 가치

포자의 표면 특성은 식별을위한 가장 신뢰할 수있는 기초를 제공합니다.환경에 의해 쉽게 영향을받는 캡의 색과는 달리, 박테리아 주름의 배열, 구멍의 구조 및 포자의 색은 종 내에서 비교적 안정적입니다.American Mycology Association의 데이터에 따르면 일반적인 버섯 종의 약 70%가 포자 표면 특성만으로 정확하게 식별 될 수 있습니다.

🛠️ 전문 장비 : 투자하기에 적합한 도구

기본 장비 목록

-10X 핸드 헬드 돋보기 유리 : 박테리아 프릴 가장자리와 작은 구멍 관찰

  • 메스 : 박테리아 폴드 부착의 세로 관찰
  • 포자 인쇄 용지 : 흑백 양면 종이 또는 알루미늄 호일
  • 디지털 카메라 : 매크로 촬영 세부 사항
  • 현장 기록 책 : 방수 용지
  • 통치자와 색상 차트

전문가들은 : 돋보기의 품질에 인색하지 마십시오.아마존 열대 우림을 방문 할 때 고품질 10 배의 돋보기 유리가 매우 유사한 두 개의 아마니타 박테리아 (하나는 식용과 다른 고유 한 독성을 구별하는 데 도움이되었습니다.

📖 포자 생성 표면의 유형 : 식별을위한 실용 가이드
📌 아가미/라멜라 : 가장 일반적인 포자 표면

구조와 기능에 대한 자세한 설명

주름은 펠렛의 중심에서 바깥쪽으로 방출 된 플라스크 구조물입니다. 이 설계는 결코 우연이 아닙니다. 표면적 대 부피 비율을 최대화하고 포자 수율을 최대화합니다. 현미경 하에서, 박테리아 주름의 표면은 수백만 포드로 덮여 있으며, 각각은 일반적으로 4 개의 포자를 생성합니다.

프론트 식별 기술

  • 관찰 각도 : 버섯을 뒤집고 아래에서 관찰하십시오.
  • 가벼운 이용 : 측면 조명을 사용하여 박테리아 주름의 배열 패턴을 관찰하십시오.
  • 터치 테스트 : 장갑을 착용 한 후 세균 주름을 터치하여 질감을 느껴보십시오.

전형적인 대표 및 실제 사례

  • Toadstools : 일반적인 식용 버섯 포함

-Amanita : 세계에서 가장 유독 한 버섯 종이 포함되어 있습니다.

  • 버섯 속 : 상업적으로 재배 된 Agaricus bisporus 포함

-Cenus Genus : 잔디밭과 열린 지역에서 일반적으로 발견됩니다.

빠른 팁 : 북미 산림에서는 주름진 오징어로 버섯을 만나면 더욱주의를 기울여야합니다.

📌 모공/튜브 : Boletus의 로고

구조적 및 기능적 깊이 분석

구멍 구조는 실제로 조밀하게 배열 된 수직 튜브의 개구부입니다. 이 튜브의 내부 벽은 포자 생성 조직으로 덮여 있습니다. 흥미로운 사실 ​​: 특정 기공 박테리아의 튜브 기공 층은 매년 업데이트되어 버섯의 나이를 결정하는 데 도움이됩니다.

Boletus 식별에 대한 실질적인 테스트

작년에 콜로라도의 로키스를 검사하는 동안, 나는 지역 주민들이 구멍의 특성을 통해 감히 모으지 않는 맛있는 볼레 투스를 정확하게 확인했습니다.주요 기능은 다음과 같습니다.

  • 구멍 크기와 모양
  • 색 변색 반응
  • 버섯과 분리의 어려움

전문 기술 : 블레이드를 사용하여 구멍의 표면을 부드럽게 긁어 내고 색상 변화를 관찰하십시오.일부 Boletus는 즉시 파란색으로 변할 것이며, 이는 중요한 식별 기능입니다.

📌 치아/가시 : 독특한 구조

현장 식별을위한 큰 포인트

곰팡이 커뮤니티에서 치아와 같은 구조는 비교적 드물기 때문에 발견되면 식별을 쉽게 좁힐 수 있습니다. Monkey Erectus는 가장 유명한 대표자입니다.안전하고 식용 일뿐 만 아니라 훌륭한 의약 가치도 있습니다.

수집 경험 : 미시간의 나무 숲에서, 나는 너도밤 나무의 죽은 나무에서 Erectus 버섯을 찾는 것이 성공률이 가장 높다는 것을 알았습니다.그들의 치아와 같은 구조는 부드러 우면 순수한 흰색이며 성숙 할 때 약간 노란색 일 수 있습니다.

📐 False Gills : Chanterelle의 독점 기능

곰팡이 주름의 실제 차이

의사-펀 스 폴드는 버섯 식별에서 가장 오해 된 기능 중 하나입니다.몬태나에서 워크숍을 이끌 때, 나는 연수생의 60% 이상이 처음에 진실과 허위 양고기 주름을 정확하게 구별 할 수 없다는 것을 알았습니다.

주요 차이

  • 분리 테스트 : 나이프 팁을 사용하여 "박테리아 주름"을 분리하십시오. 곰팡이 주름은 분리하기 쉽고 의사 박테리아 및 곰팡이 살이 통합됩니다.
  • 두께 관찰 : Pseudobacteria fold는 더 두껍고 더 둔합니다.
  • 분기 패턴 : 의사- 펀 스 폴드에는 종종 불규칙한 분기가 있습니다

식용 안전 팁 : 실제 Cantharellus는 세계에서 가장 안전한 야생 식용 곰팡이 중 하나이며, 곰팡이 주름을 가진 위조자는 독성이있을 수 있습니다.이 차이는 삶과 죽음의 차이 일 수 있습니다.

📌 Smooth : 드문 포자 생산 방법

비교적 희귀하지만 부드러운 포자 생산 표면은 일부 그룹에서 일반적입니다.이 버섯은 일반적으로 비가 튀거나 곤충과 같은 다른 수단을 통한 포자 전송을 보장합니다.

🍄 박테리아 주름 특성에 대한 자세한 설명 : 이론에서 실습까지
📌 박테리아 주름 부착 방법 : 가장 중요한 식별 기능

내가 가르친 버섯 식별 과정에서 곰팡이 폴드를 부착하는 방법은 내가 강조한 첫 번째 교훈이었습니다.이 기능의 중요성은 과장 될 수 없습니다. 식별 범위를 여러 속으로 즉시 좁힐 수 있습니다.

실제 관찰 방법

1. 대표 표본 선택 : 각 연령에 대해 하나를 선택하십시오

2. 정확한 절단 : 날카로운 칼날을 사용하여 줄기의 중간 선을 따라 세로 절단

3. 관찰 각도 : 좋은 빛의 여러 각도에서 관찰

4. 결과 기록 : 현장 레코드 북에서 즉시 그리거나 설명하십시오.

무료 (무료) frit의 식별 가치

특발성 주름은 Amanita 속과 버섯 속의 주요 특징입니다.케라틴이 케라틴과 전혀 접촉하지 않는다는 것을 알게되면이 두 가지 중요한 속의 구성원이 직면하게 될 것입니다.현재 다른 기능을 즉시 확인해야합니다.

-Amanita : 보통 박테리아 관과 고리가 있습니다

  • 버섯 속 : 실제 멸균이 없습니다

전문가들은 : 버섯을 세로로 자르면 절단 표면을 깔끔하게 유지하고 미세한 구조를 압박하고 파괴하지 않도록주의하십시오.수술 블레이드를 사용하는 것이 좋습니다. 일반 칼보다 정확합니다.

📌 박테리아 주름 간격 : 정량적 관찰 기술

많은 야외 가이드는 단순히 "집중적"또는 "스파스"로 묘사되지만 학생들은 정량적 접근 방식을 취하도록 가르칩니다.

표준 측정 프로토콜

1. 캡 가장자리에서 대표 영역을 선택하십시오.

2. 통치자를 사용하여 길이 1cm를 측정하십시오

3.이 길이 내에서 완전한 박테리아 주름의 수를 계산합니다.

4. 캡의 다른 위치에서 측정을 반복하십시오

간격 분류 표준

  • 밀도 : 15 개 이상의 박테리아가 센티미터를 접습니다
  • 보통 : 8-15 박테리아는 센티미터를 접습니다
  • 스파스 : 8 % 미만의 박테리아가 백분율로 접 힙니다

실용적인 응용 : 워싱턴 주 캐스케이드 산맥에서 나는 정확한 수의 주름 (22 피트 %)을 가진 희귀 한 락토 바실러스를 확인하는 데 도움을주었습니다.

📌 박테리아 주름 두께 및 질감 : 촉각 식별 기술

두께 측정 표준

  • 좁은 : 3mm 미만
  • 중간 : 3-5 mm
  • 너비 : 5 mm보다 큽니다

텍스트 테스트 안전 프로토콜

버섯이 높은 독성의 위험이 없음을 확인하기 전에 직접 피부 접촉을 피하십시오. 다음 방법을 사용할 수 있습니다.

1. 깨끗한 나이프 팁을 사용하여 펠릿을 누릅니다.

2. 파괴 방법을 관찰하십시오

3. 칼 끝을 사용하여 작은 조각을 골라 관찰하십시오.

텍스트 유형 및 식별 가치

  • Crisp : Lactiflora와 Red Mushrooms의 전형적인 특성, 가벼운 압박 후 작은 조각으로 분쇄
  • 왁스 : 왁스 속의 독특한 특징, 문지르면 맑은 왁스 느낌
  • 강인함 : 대부분의 리그닌 박테리아의 특성은 찢기 어렵습니다.
📌 박테리아 주름 색상 : 동적 변화에 대한 식별 단서

주름의 색은 포자가 성숙함에 따라 변화하기 때문에 아마도 가장 오해 된 기능 일 것입니다.시스템 관찰 방법을 개발했습니다.

색상 관찰 계약

1. 다른 발달 단계에서 표본을 수집하십시오

2. 초기 색상을 즉시 기록하십시오 (현장 조건 하에서)

3. 최종 색상을 확인하기 위해 포자 인쇄물을 만드십시오

4. 환경 요인이 색상에 미치는 영향에주의를 기울이십시오.

일반적인 색상 진화 모드

  • 버섯 : 흰색 → 핑크 → 초콜릿 브라운

-Genus 속 : 흰색 → 핑크 → 검은 색

-Amanita : 보통 흰색 또는 약간 노란색을 유지합니다

일반적인 실수 방지 : 메모리만으로는 색상을 판단하지 말고 표준 색상 차트를 사용하십시오.조명 조건은 색상 인식에 크게 영향을 줄 수 있습니다.

🛠️ 구멍의 특성 (튜브 홀) : Boletus 전문가의 관점
📌 기공 크기 및 모양 : 정확한 측정 방법

기공 크기 분류

  • 큰 구멍 : 1mm보다 큰 직경, 육안으로 명확하게 보입니다.
  • 작은 구멍 : 직경이 0.5-1 mm, 돋보기를 관찰해야합니다.
  • 마이크로 포어 : 직경은 0.5mm 미만이고 표면은 매끄럽게 보입니다.

전문 측정 기술

밀리미터 당 홀 수의 측정 방법을 사용하는 것이 좋습니다.

1. 대표 영역을 선택하십시오

2. 돋보기와 미세 눈금자를 사용하십시오

3. 길이 1mm 이내의 구멍 수를 계산하십시오.

4. 전환 공식 : 기공 크기 ≈1/밀리미터 당 기공 수

실제 케이스 : 맛있는 볼레 테스를 식별 할 때 구멍의 크기는 주요 기능 중 하나입니다.진짜 맛있는 볼 레트는 일반적으로 밀리미터 당 2-3 개의 구멍이 있습니다.

📌 색상 변화 반응 : 화학 식별의 비밀

일부 볼레 투스의 모공은 압축 후 색이 변할 것입니다. 이는 페놀 화합물의 산화로 인한 것입니다.이 반응은 매우 중요한 식별 기능입니다.

표준 테스트 방법

1. 부드럽고 온전한 구멍 표면을 선택하십시오

2. 칼이나 손톱의 뒷면으로 부드럽게 누릅니다.

3. 즉시 타이밍을 시작하십시오

4. 색상 변화 속도, 최종 색상 및 복구 시간 기록

일반적인 색상 변색 유형

  • 파란색 변화 : 특정 boletus 박테리아의 특성, 반응 속도는 즉시 다릅니다.
  • 브라우닝 : 더 일반적인 반응, 식별 값이 적습니다
  • 빨간색 변경 : 비교적 드물지만 식별 가치가 높습니다.

안전 팁 : 색상 변색 반응은 독성 지표가 아닙니다.일부 파란색 형태의 볼레 투스는 식용이며 일부는 변색 대신 독성입니다.

🍄 포자 인쇄 생산 : 가정 실험실 기술

포자 인쇄는 버섯 식별에서 가장 신뢰할 수있는 기능 중 하나입니다.20 년 이상 개선 된 후 포자 인쇄를 만드는 가장 효과적인 방법을 요약했습니다.

전문 포자 인쇄 생산 키트

  • 흑백 양면 포자 인쇄지
  • 투명 용기 또는 유리 컵

  • 멸균 메스
  • 태그 시스템

생산 공정 최적화

1. 방금 완전히 확장 된 멸균 커버를 선택하십시오.

2. 멸균 조건에서 멸균 된 줄기 제거

3. 캡 포자를 종이에 내려 놓습니다.

4. 증류수를 가볍게 뿌려 습도를 높입니다

5. 직접 밀봉을 피하기 위해 컨테이너로 덮으십시오

6. 15-20 ° C에서 숙박하십시오

시간 가이드

  • 버섯 : 4-8 시간
  • 구멍 버섯 : 12-24 시간
  • 노인 표본 : 2-4 시간이면 충분할 수 있습니다

전문가 팁 : 종이 대신 알루미늄 호일을 사용하면 포자 축적의 두께와 색상을 동시에 관찰하여 추가 정보를 제공 할 수 있습니다.

🍄 실제 신청 사례
📌 사례 1 : 유독 한 Amanita와 식용 버섯 속을 구별

이것은 북미에서 가장 중요한 식별 차이 중 하나입니다.

장례식

  • 박테리아는 자유롭게 접습니다
  • 젊고 성숙한 초콜릿 브라운의 분홍색
  • 포자 인쇄 진한 갈색
  • 실제 멸균 스텐트가 없습니다

Amanita 특성

  • 박테리아는 무료 또는 epiphytic을 접습니다
  • 보통 흰색 또는 가벼운 상태로 유지됩니다
  • 포자 인쇄 흰색
  • 박테리아 트렌치와 박테리아 고리가 있습니다

삶과 죽음의 차이 :이 차이는 당신의 생명을 구할 수 있습니다.박테리아 트로프와 포자 인쇄 흰색으로 버섯을 수집하지 마십시오.

📌 사례 2 : Boletus의 식용 평가

록키 산맥의 서쪽 부분에서 나는 학생들에게 다음 과정을 사용하여 볼레 투스를 평가하도록 가르칩니다.

1. 구멍 특성이 Boletaceae과와 일치하는지 확인

2. 색상 변화 반응 테스트

3. 곤충 침입 여부 확인(곤충을 기피하는 종은 주의가 필요함)

4. 미각검사(작은 조각을 씹어 뱉어낸다)

중요한 안전 경고: 숙련된 채집자라도 포르치니 버섯에는 주의해야 합니다.일부 종은 위장 장애를 일으킬 수 있습니다.

📌 사례 3: 살구류와 위조품 구별하기

태평양 북서부에서는 실제 살구류와 독성 모조품을 구별하는 것이 중요합니다.

진짜 살구

  • 가짜 아가미: 두껍고, 뭉툭하며, 종종 두 갈래로 갈라집니다.
  • 색상 : 균일한 황금색 노란색
  • 냄새 : 신선한 살구 향
  • 성장 모드: 단일 또는 소그룹

가짜 살구류

  • 곰팡이 주름: 얇고 날카로우며 두 갈래로 갈라지지 않음
  • 색상: 오렌지-레드 톤
  • 냄새 : 특별한 향이나 불쾌한 냄새는 없습니다.
  • 성장 패턴: 일반적으로 큰 클러스터에서

📌 환경적 요인의 영향

포자 표면 특성은 환경 조건의 영향을 받습니다.높은 고도에서는 자외선으로 인해 색상이 더 진해질 수 있습니다.습한 환경에서는 아가미가 더 조밀해 보일 수 있습니다.

장기 모니터링 프로젝트를 진행하는 동안 일부 종의 포자 표면 특성이 계절에 따라 변하는 것을 발견했습니다.봄 표본은 일반적으로 아가미가 더 촘촘하고 색상이 더 밝습니다.

📌 지리적 변화

동일한 종이라도 지리적 위치에 따라 미묘한 차이를 보일 수 있습니다.서해안의 아마니타는 동해안의 아마니타보다 주름이 약간 더 촘촘할 수 있습니다.
